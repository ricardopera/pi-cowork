# Funnel Analysis

When asked to analyze a funnel:

1. Define the funnel steps from entry to the target outcome.
2. Compute step-to-step conversion and cumulative conversion.
3. Identify the biggest drop-off step and quantify it.
4. Form hypotheses for the drop-off; suggest tests to validate.
5. Export as create_xlsx (steps + rates) and create_docx (findings).
