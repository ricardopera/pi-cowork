# Q&A / FAQ Prep

When asked to build an FAQ or Q&A set:

1. Read the source material (product docs, briefing, transcript).
2. Identify the likely audience's real questions (practical, not generic).
3. Write each Q concisely; answer in 1-3 sentences with specifics.
4. Group by theme; flag questions you can't answer from the source.
5. Export as create_docx or create_file.
