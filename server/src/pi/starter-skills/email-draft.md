# Draft an Email

When asked to draft or reply to an email:

1. Clarify recipient, goal, and tone with ask_question if not stated.
2. Plan the structure: subject line, greeting, body, call-to-action, sign-off.
3. Keep it concise — one main idea per paragraph; lead with the ask.
4. Match the requested tone (formal / friendly / urgent).
5. Present the draft as text; offer to save it as a .docx or copy to clipboard.
