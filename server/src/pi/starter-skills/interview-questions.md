# Interview Questions

When asked to write interview questions:

1. Clarify the role, level, and which signal each question probes (skill/behavior/values).
2. Mix: technical, behavioral (STAR), and situational.
3. Make questions open-ended; add follow-ups and what "good" looks like.
4. Avoid brainteasers and biased/irrelevant questions.
5. Export a create_docx guide per interviewer.
