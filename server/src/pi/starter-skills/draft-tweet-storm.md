# Write a Tweet Thread

When asked to write a thread:

1. Hook in post 1 (the payoff or the question); number subsequent posts.
2. One idea per post; keep each scannable with line breaks.
3. Build a logical arc: setup → evidence → takeaway → CTA.
4. Reference/quote sources; keep within platform limits.
5. Present the thread as numbered text; offer create_file export.
