# Risk Register

When asked to build a risk register:

1. Brainstorm risks across categories (technical, schedule, org, external).
2. Score each on likelihood × impact; rank by exposure.
3. Assign an owner and a mitigation/contingency per top risk.
4. Note trigger conditions and review cadence.
5. Export as create_xlsx; revisit at each milestone.
