# Meeting Notes

When asked to produce meeting notes/minutes:

1. Ask for the source (transcript file, raw notes, or pasted text) and key attendees if unclear.
2. Extract: date, attendees, agenda items, decisions, action items (owner + due).
3. Structure as: Summary → Decisions → Action Items → Open Questions.
4. Keep action items explicit and assignable ("Owner: X — Do Y by Z").
5. Export as create_docx or create_file; offer to save durable context to memory.
