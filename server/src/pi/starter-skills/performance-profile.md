# Performance Profile

When asked to investigate performance:

1. Establish a reproducible measurement (before optimizing).
2. Profile to find the actual bottleneck (don't guess).
3. Form a hypothesis; change one thing; re-measure.
4. Report: baseline, change, result, and cost/tradeoff.
5. Note regressions or areas not yet covered.
