# Translate / Localize

When asked to translate or localize:

1. Confirm source language, target language, and tone/register.
2. Preserve meaning and intent — adapt idioms, don't translate literally.
3. Keep formatting (markdown, placeholders) intact.
4. Flag ambiguous terms and offer alternatives.
5. Present the translation; offer create_file export.
