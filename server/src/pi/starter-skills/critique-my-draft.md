# Critique a Draft

When asked to critique a draft:

1. Read the whole thing first; understand the author's intent.
2. Lead with what's working (specific, not generic praise).
3. Group issues: structure, clarity, accuracy, completeness.
4. Make each suggestion actionable ("move X before Y", not "reorganize").
5. Distinguish must-fix from nice-to-have; offer to apply changes.
