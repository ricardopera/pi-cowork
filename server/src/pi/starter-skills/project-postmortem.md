# Project Postmortem

When asked to write a postmortem/RCA:

1. State the impact (what, who, how long) factually and without blame.
2. Build a timeline from logs/notes; mark the trigger and detection.
3. Identify root cause(s) and contributing factors.
4. Capture action items with owners: fix, detect, prevent.
5. Export as create_docx; save durable lessons to memory.
