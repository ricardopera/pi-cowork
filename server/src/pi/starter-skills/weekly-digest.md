# Weekly Status Update

When asked to write a status update:

1. Gather inputs (commits, closed tickets, notes) for the period.
2. Structure: Shipped / In progress / Blocked / Next week.
3. Be concrete and specific; call out blockers with owners.
4. Keep it skimmable — bullets, not prose.
5. Export as create_markdown or paste-ready text.
