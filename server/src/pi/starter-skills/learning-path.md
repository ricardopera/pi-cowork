# Build a Learning Path

When asked to build a learning path:

1. Clarify the learner's level and goal (and time budget).
2. Sequence fundamentals → core skills → advanced → project.
3. For each stage: concepts, a hands-on exercise, and a check-for-understanding.
4. Recommend free, high-quality resources (with links).
5. Export as create_markdown or create_docx.
