# Press Release

When asked to write a press release:

1. Lead with the news in the first sentence (who/what/why/when).
2. Follow the inverted pyramid: most important → supporting → boilerplate.
3. Include a quote and a clear "About" paragraph.
4. Add contact info and a ### end marker.
5. Export as create_docx; keep under one page.
