# Build a Data Dictionary

When asked to build a data dictionary:

1. List every field with: name, type, description, allowed values/range, nullability.
2. Note the source of each field and any transformations applied.
3. Flag derived fields with their formula.
4. Flag PII/sensitive fields with handling rules.
5. Export as create_xlsx for easy reference.
