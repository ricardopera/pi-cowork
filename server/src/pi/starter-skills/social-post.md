# Write a Social Post

When asked to write a social post:

1. Clarify platform (X/LinkedIn/etc.), goal, tone, and length limits.
2. Write a hook in the first line; keep it scannable.
3. Match platform norms (hashtags/mentions, formality).
4. Offer 2-3 variants; present as text or create_file.
5. Optionally schedule via the scheduler if the user wants it posted later.
