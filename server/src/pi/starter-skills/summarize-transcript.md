# Summarize a Transcript

When asked to summarize a transcript/recording:

1. Read the transcript (read tool, or fetch captions).
2. Identify speakers, agenda, decisions, and action items.
3. Produce: 3-sentence summary → key decisions → action items (owner+due) → open questions.
4. Attribute statements to speakers where it matters.
5. Offer create_docx export; save durable decisions to memory.
