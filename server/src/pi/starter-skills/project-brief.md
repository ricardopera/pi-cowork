# Project Brief

When asked to write a project brief/charter:

1. Clarify the project name, sponsor, and primary goal with ask_question.
2. Capture: Problem/Opportunity, Goals (SMART), Scope (in/out), Stakeholders, Milestones, Risks, Success metrics.
3. Keep each section tight — a brief should fit on one page.
4. Export as create_docx; present_files.
