# Draft a Roadmap

When asked to draft a roadmap:

1. Clarify horizon, themes, and constraints (capacity, dependencies).
2. Group initiatives into Now / Next / Later (or by quarter).
3. For each: outcome, owner, dependencies, rough sizing.
4. Sequencing > dates; show dependencies.
5. Export as create_xlsx or create_docx with a simple timeline.
