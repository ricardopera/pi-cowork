# Analyze Survey Results

When asked to analyze survey/poll data:

1. Load the responses (read/CSV); note sample size and missing data.
2. For quantitative questions: compute counts, percentages, averages.
3. For open-ended: group into themes with representative quotes.
4. Highlight the top 3-5 findings and any notable segments.
5. Export a create_xlsx summary and a create_docx narrative.
