# Troubleshoot an Issue

When asked to debug/troubleshoot:

1. Reproduce: get the exact steps, inputs, and full error message.
2. Form a hypothesis; check logs/files with read/grep/bash.
3. Change ONE thing at a time; verify after each.
4. When fixed, explain root cause and the fix; note how to prevent recurrence.
5. Save durable lessons (env quirks, gotchas) to memory.
