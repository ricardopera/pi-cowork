# Travel Itinerary

When asked to build a travel itinerary:

1. Collect flights, hotels, meetings, and addresses.
2. Order chronologically with local times + timezones.
3. Add confirmation numbers, addresses, and contact numbers.
4. Include buffer time and notes (visa, transport, essentials).
5. Export as create_docx; offer a calendar-friendly create_file version.
