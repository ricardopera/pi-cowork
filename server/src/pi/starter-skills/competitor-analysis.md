# Competitor Analysis

When asked to analyze competitors:

1. Clarify the subject company, the competitors to compare, and dimensions (price, features, audience).
2. Research each via browser tools (navigate/scrape their sites, press, reviews).
3. Build a comparison matrix (features × competitors) using todo_write to track.
4. Note strengths, weaknesses, and positioning for each.
5. Present a comparison table as create_xlsx and a narrative as create_docx.
6. Cite sources; flag uncertainty where data is thin.
