# Customer Support Reply

When asked to draft a support reply:

1. Acknowledge the issue and the customer's goal; show you read it.
2. Give the answer directly first, then details; avoid jargon.
3. If you can't fix it, say what you CAN do and next steps with a timeline.
4. Match tone to the situation (urgent vs. casual); proofread.
5. Offer to save recurring answers to memory for reuse.
