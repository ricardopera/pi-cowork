# Write a Cron Expression

When asked to write a cron expression:

1. Clarify the schedule (frequency, timezone, gaps for nights/weekends).
2. Provide the 5-field expression; explain each field.
3. Give 3 natural-language examples of when it fires.
4. Warn about common pitfalls (day-of-month vs day-of-week interaction, DST).
5. Offer to create a scheduled task via the scheduler.
