# Write OKRs

When asked to write OKRs:

1. Clarify the timeframe and owner (team/individual).
2. Define 2-4 Objectives (qualitative, inspirational, achievable).
3. For each, 3-5 Key Results (measurable, time-bound, outcome-focused).
4. Avoid task-list KRs; prefer metrics (%, counts, $).
5. Export as create_xlsx or create_docx.
