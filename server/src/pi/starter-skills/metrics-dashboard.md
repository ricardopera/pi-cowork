# Build a Metrics Dashboard

When asked to build a metrics dashboard:

1. Confirm the audience and the 4-6 KPIs that matter to them.
2. For each KPI: definition, data source, refresh cadence, target.
3. Build a live create_artifact (HTML) rendering the KPIs with clear visuals.
4. Use simple, honest chart types; label axes and units.
5. Note what's NOT included and why (avoid vanity metrics).
