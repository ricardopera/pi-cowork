# Write a Bug Report

When asked to write a bug report:

1. Capture: title, severity, environment (OS/browser/version), build.
2. Write exact reproduction steps (numbered, copy-pasteable).
3. State expected vs. actual behavior.
4. Attach logs/screenshots/error text verbatim.
5. Note suspected cause if known; mark as needs-triage otherwise.
6. Export as create_markdown or paste-ready text.
