# FAQ Page

When asked to build a FAQ/help page:

1. Source real questions (tickets, chats, search logs); rank by frequency.
2. Group into categories; write each answer short and actionable.
3. Link to deeper docs; cross-link related questions.
4. Keep a consistent voice and a "still need help?" path.
5. Export as create_markdown or create_file (HTML).
