# Onboarding Doc

When asked to write an onboarding/setup guide:

1. Clarify the audience (new hire, new tool user) and their starting point.
2. Structure: Prerequisites → Step-by-step setup → First tasks → Where to get help.
3. Number every step; include exact commands and expected results.
4. Call out common pitfalls explicitly.
5. Export as create_markdown or create_docx; verify steps are runnable.
