# Newsletter Draft

When asked to draft a newsletter:

1. Confirm audience, sections, and length.
2. Write a short intro framing the issue; lead with the strongest item.
3. Keep each item to 2-3 lines + a link; group by theme.
4. End with a clear call-to-action.
5. Export as create_markdown or create_file (HTML).
