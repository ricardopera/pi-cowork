# Make a Diagram

When asked to make a diagram:

1. Clarify what to convey (sequence, architecture, state, flow).
2. Pick the right Mermaid diagram type (flowchart/sequence/C4/ERD).
3. Keep boxes/labels short; show only the relevant relationships.
4. Add a one-line caption explaining how to read it.
5. Embed in a create_markdown or a live create_artifact.
