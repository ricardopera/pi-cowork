# Write User Stories

When asked to write user stories:

1. Frame each as "As a <persona>, I want <goal> so that <benefit>."
2. Add 2-5 acceptance criteria per story (Given/When/Then).
3. Keep stories small enough to complete in a sprint; split large ones.
4. Note dependencies and rough sizing.
5. Export as create_xlsx (backlog) or create_markdown.
