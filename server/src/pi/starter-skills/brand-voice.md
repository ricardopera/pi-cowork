# Brand Voice Guide

When asked to define/apply brand voice:

1. Capture 3-5 voice attributes (e.g., clear, warm, confident) with do/don't examples.
2. Specify tone variations by context (marketing vs. support vs. error).
3. List vocabulary rules (preferred/avoided terms, capitalization, numbers).
4. Provide before/after rewrites demonstrating the voice.
5. Export as create_markdown; save to memory so future drafts follow it.
