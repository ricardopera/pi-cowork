# Write an Announcement

When asked to write an announcement:

1. Lead with the news and what it means for the reader (WIIFM).
2. Give the key details; keep jargon out; use short paragraphs.
3. Address the obvious follow-up questions (timing, impact, action needed).
4. End with where to get help or learn more.
5. Offer a long + short variant; export as create_markdown or create_docx.
