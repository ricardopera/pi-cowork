# Prep for a Meeting

When asked to prep for a meeting:

1. Clarify the meeting type (decision / update / 1:1) and your goal.
2. Summarize relevant context: prior decisions, open items, metrics.
3. List the 2-3 points you must land; anticipate questions with answers.
4. Note any asks and what you'll commit to.
5. Export a one-page create_docx brief.
