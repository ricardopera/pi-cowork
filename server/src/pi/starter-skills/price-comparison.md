# Compare Options

When asked to compare options/alternatives:

1. Confirm the options and the evaluation criteria.
2. Weight criteria by importance (1-5); score each option per criterion.
3. Compute weighted totals; note close calls.
4. Highlight the recommendation but show the math so it's auditable.
5. Export as create_xlsx with the scoring table.
