# Compliance Mapping

When asked to map compliance:

1. List the framework's requirements; for each, link your control/evidence.
2. Mark status: Implemented / Partial / Gap.
3. For gaps, note the remediation owner and target date.
4. Keep evidence pointers (doc links, config, ticket numbers).
5. Export as create_xlsx matrix + create_docx narrative for auditors.
