# Build User Personas

When asked to build personas:

1. Ground each in real signal (interviews, analytics), not stereotypes.
2. Capture: role, goals, pain points, jobs-to-be-done, a typical day snippet.
3. Keep to 2-4 personas that meaningfully change what you'd build.
4. Note what each persona is NOT, to avoid overlap.
5. Export as create_docx; offer to save key traits to memory.
