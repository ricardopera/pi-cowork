# Pricing Page Copy

When asked to write pricing copy:

1. Clarify the plans, prices, and what's in each tier.
2. Lead with the recommended plan; anchor pricing clearly.
3. Use parallel feature bullets; mark included vs. add-on consistently.
4. Anticipate the top 3 objections in a short FAQ.
5. Export as create_markdown or create_file (HTML).
