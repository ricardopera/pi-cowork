# Write a Contributing Guide

When asked to write CONTRIBUTING:

1. Cover: how to report bugs, request features, set up the dev environment.
2. Explain the PR workflow (branch naming, tests, review).
3. State coding standards and commit-message conventions.
4. Include a code of conduct pointer.
5. Export as create_markdown (CONTRIBUTING.md).
