# Rewrite for Tone

When asked to rewrite for tone:

1. Confirm the target tone/level and any constraints (length, banned words).
2. Preserve the original meaning and key facts — only style changes.
3. Make systemic changes (sentence length, vocabulary, voice) not line-edits.
4. Provide before/after so the change is auditable.
5. Offer 2 variants if the tone is ambiguous; save the chosen style to memory.
