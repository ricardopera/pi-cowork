# Draft a Policy

When asked to write a policy:

1. State purpose and who it applies to.
2. Write the rules as clear, enforceable statements.
3. Include definitions, exceptions, and the owner/approver.
4. Add effective date + review cadence.
5. Export as create_docx; keep it short enough to actually read.
