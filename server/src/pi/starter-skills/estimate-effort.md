# Effort Estimate

When asked to estimate effort:

1. Break the work into a WBS down to ~1-day tasks.
2. Estimate each (best/likely/worst); sum the likely, note the range.
3. Call out the riskiest/most-uncertain tasks explicitly.
4. State assumptions (availability, dependencies, definition of done).
5. Export as create_xlsx (tasks + hours) + create_docx (assumptions).
