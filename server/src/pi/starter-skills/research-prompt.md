# Write a Prompt

When asked to write or improve an LLM prompt:

1. Clarify the task, model, and desired output format.
2. Structure: role/context → task → constraints → output format → examples.
3. Be specific and imperative; avoid ambiguity.
4. Include 1-2 concrete examples where helpful.
5. Present the prompt in a code block; offer variants (concise vs. detailed).
