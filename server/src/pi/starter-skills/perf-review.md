# Performance Review

When asked to write a performance review:

1. Gather signal: outcomes, metrics, peer feedback, prior goals.
2. Structure: Summary → Strengths → Growth areas → Goals.
3. Be specific and evidence-based; avoid halo/horns bias.
4. Separate the person from the work; lead with actionable coaching.
5. Export as create_docx.
