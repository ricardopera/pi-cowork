# Stakeholder Map

When asked to map stakeholders:

1. List everyone affected by or influential over the work.
2. Score influence × interest; place on a 2x2 (manage closely / keep satisfied / inform / monitor).
3. Define the engagement approach + cadence per quadrant.
4. Note each stakeholder's primary concern and what "success" looks like for them.
5. Export as create_xlsx matrix.
