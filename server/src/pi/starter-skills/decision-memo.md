# Decision Memo

When asked to write a decision memo/ADR:

1. State the decision needed and context in 2-3 sentences.
2. List 2-4 options; for each: pros, cons, cost/risk.
3. Give a clear recommendation with reasoning.
4. Note what would change the recommendation.
5. Export as create_docx or create_markdown.
