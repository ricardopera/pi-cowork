# Cohort Analysis

When asked to run a cohort analysis:

1. Define the cohort grouping (signup period, segment) and the event of interest.
2. Compute the metric per cohort per period (e.g., % active in week N).
3. Build a cohort triangle (cohort × period) as create_xlsx.
4. Read the patterns: which cohorts retain better, where drop-off happens.
5. Suggest hypotheses for the differences; flag data quality issues.
