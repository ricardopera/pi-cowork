# Extract a Table

When asked to extract a table:

1. Locate the source (PDF/image/HTML/text).
2. Identify rows and columns; note headers and units.
3. Transcribe faithfully; flag merged cells or ambiguity rather than guessing.
4. Validate: row count, totals where checkable.
5. Export as create_xlsx with the headers and rows.
