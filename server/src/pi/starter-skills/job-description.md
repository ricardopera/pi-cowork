# Write a Job Description

When asked to write a job description:

1. Clarify role title, level, team, and 3-5 must-have requirements.
2. Structure: About the role → Responsibilities → Requirements → Nice-to-haves → How to apply.
3. Use inclusive, bias-aware language; avoid unnecessary must-haves.
4. Keep it skimmable with short bullets.
5. Export as create_docx.
