# Write a Blog Post

When asked to write a blog post:

1. Clarify audience, target length, tone, and key takeaway.
2. Write a compelling title + hook; outline sections with todo_write.
3. Draft section by section; use concrete examples and short paragraphs.
4. End with a clear takeaway or call-to-action.
5. Export as create_markdown or create_docx; offer to schedule via the scheduler.
