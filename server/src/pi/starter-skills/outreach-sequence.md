# Draft a Sales Outreach

When asked to write sales outreach:

1. Clarify the prospect persona, value prop, and goal (meeting/reply).
2. Write a 3-touch sequence: opener (value, personalized), nudge, break-up.
3. Keep each under 120 words; one clear CTA; credible specificity.
4. Avoid generic templates — reference the prospect's context.
5. Present as text; offer to save as create_docx.
